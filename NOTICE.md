APRS Net Android - third-party attributions
==============================================

APRS symbol graphics
-----------------------
The 24x24 PNG sprite sheets in `app/src/main/res/drawable/`:

  - aprs_symbols_24_0.png    (primary table)
  - aprs_symbols_24_1.png    (secondary / alternate table)
  - aprs_symbols_24_2.png    (overlay characters)

are taken from the aprs.fi high-resolution APRS symbol set by Heikki
Hannikainen, OH7LZB:

  https://github.com/hessu/aprs-symbols

The set is provided for free use by APRS applications. Source-and-copyright
information for individual symbols is in the upstream COPYRIGHT.md.

osmdroid + osmbonuspack
---------------------------
Map tiles are rendered by osmdroid (Apache 2.0). Marker clustering uses
osmbonuspack's RadiusMarkerClusterer.

  https://github.com/osmdroid/osmdroid
  https://github.com/MKergall/osmbonuspack
